# 🚀 智备小语 - 免费部署上线指南

> 目标：将系统部署到 Render 免费平台，获得公共域名，他人扫码即可使用
>
> ⚠️ 由于 GitHub 在国内可能无法访问，本指南使用 **Gitee** 替代

---

## 📋 前置准备

| 项目 | 说明 |
|------|------|
| Gitee 账号 | 去 [gitee.com](https://gitee.com) 注册（国内平台，秒开） |
| Render 账号 | 用邮箱注册 [render.com](https://render.com) |
| DeepSeek API Key | 已有的 Key，配置到服务器供所有人共用 |

---

## 第一步：在 Gitee 创建仓库

1. 登录 https://gitee.com
2. 点击右上角 **"+"** → **"新建仓库"**
3. 填写信息：
   - **仓库名称**: `zhibeixiaoyu`
   - **路径**: 自动填充
   - **开源**: 选择 **"公开"**（Render 需要公开仓库才能拉取代码）
   - **不要** 勾选 "使用 Readme 初始化仓库"
4. 点击 **"创建"**

---

## 第二步：推送代码到 Gitee

打开 PowerShell，执行以下命令（**替换 `你的Gitee用户名`**）：

```bash
cd D:\dachuang\zhibeixiaoyu

D:\dachuang\tools\Git\cmd\git.exe remote add gitee https://gitee.com/你的Gitee用户名/zhibeixiaoyu.git

D:\dachuang\tools\Git\cmd\git.exe push -u gitee main
```

> 如果提示输入用户名密码，输入 Gitee 的账号和密码即可

---

## 第三步：在 Render 上部署

1. 打开 https://dashboard.render.com
2. 点击 **"New +"** → 选择 **"Web Service"**
3. 选择 **"Public Git repository"**（公开 Git 仓库）
4. 输入你的 Gitee 仓库 URL：
   ```
   https://gitee.com/你的Gitee用户名/zhibeixiaoyu.git
   ```
5. 配置服务：

| 配置项 | 填写内容 |
|--------|----------|
| **Name** | `zhibeixiaoyu` |
| **Runtime** | `Python 3` |
| **Build Command** | `pip install -r zhibei_backend/requirements.txt` |
| **Start Command** | `cd zhibei_backend && python run_prod.py` |
| **Instance Type** | `Free` |

6. 添加环境变量（点击 "Advanced" → "Add Environment Variable"）：

| Key | Value |
|-----|-------|
| `DEEPSEEK_API_KEY` | 你的 DeepSeek API Key |
| `DEFAULT_AI_PROVIDER` | `deepseek` |
| `DEFAULT_AI_MODEL` | `deepseek-chat` |
| `RENDER` | `true` |

7. 点击 **"Create Web Service"**
8. 等待部署完成（首次约 3-5 分钟）

---

## 第四步：获取公共域名

部署成功后，Render 会分配一个域名：

```
https://zhibeixiaoyu.onrender.com
```

你可以在 Render 控制台的服务页面顶部看到这个域名。

> 💡 免费版首次访问可能有约 30 秒冷启动延迟，之后访问会很快。

## ⚠️ 重要注意事项

### 关于 Render 免费版的限制

| 限制 | 说明 |
|------|------|
| 冷启动 | 15分钟无访问后服务休眠，下次访问需等约30秒 |
| 月度时长 | 750小时/月（足够24小时运行） |
| 数据库 | 每次重新部署后 /tmp 数据会丢失（SQLite 会自动重建） |
| 带宽 | 100GB/月（足够正常使用） |

### 关于 API 费用

所有用户共享你的 DeepSeek API Key，产生的 API 调用费用由你承担。
- DeepSeek Chat 模型价格约 ¥1/百万输入token，非常便宜
- 建议关注用量：https://platform.deepseek.com/usage

### 关于数据安全

- Gitee 仓库设为公开（Render 要求），但 API Key 不会出现在代码中
- API Key 仅存储在 Render 的环境变量中，是加密的

## 🔧 常见问题

### Q: 部署失败，提示 "ModuleNotFoundError"
检查 `requirements.txt` 是否包含所有依赖，确认 Build Command 路径正确。

### Q: 网页打开但功能不正常
- 打开浏览器开发者工具（F12）→ Console 查看错误
- 确认环境变量 `DEEPSEEK_API_KEY` 已正确配置

### Q: 如何更新代码？
```bash
cd D:\dachuang\zhibeixiaoyu
D:\dachuang\tools\Git\cmd\git.exe add -A
D:\dachuang\tools\Git\cmd\git.exe commit -m "更新说明"
D:\dachuang\tools\Git\cmd\git.exe push gitee main
```
然后在 Render 控制台点击 "Manual Deploy" 重新部署。

### Q: 访问太慢怎么办？
考虑升级到 Render 付费版（$7/月），或使用 Railway（$5/月免费额度）。
