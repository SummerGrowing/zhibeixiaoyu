| 项目 | 说明 |
|------|------|

## 第一步：安装 Git

1. 打开浏览器访问：https://git-scm.com/download/win
2. 下载并安装，安装时全部点"下一步"默认即可
3. 安装完成后，**重启终端/PowerShell**
4. 验证安装：
   ```bash
   git --version
   ```
   应显示类似 `git version 2.47.x.windows.x`

---

## 第二步：初始化本地 Git 仓库

打开 PowerShell，执行以下命令：

```bash
cd D:\dachuang\zhibeixiaoyu

# 初始化仓库
git init

# 添加所有文件
git add -A

# 首次提交
git commit -m "智备小语 v1.0 - 初始部署版本"
```

---

## 第三步：创建 GitHub 仓库并推送

1. 打开 https://github.com/new
2. 填写信息：
   - **Repository name**: `zhibeixiaoyu`
   - **可见性**: 选择 **Private**（私有，避免泄露 API Key 配置信息）
   - **不要** 勾选 "Add a README file"
3. 点击 "Create repository"
4. 回到 PowerShell 执行：

```bash
cd D:\dachuang\zhibeixiaoyu

# 添加远程仓库（替换 YOUR_USERNAME 为你的 GitHub 用户名）
git remote add origin https://github.com/YOUR_USERNAME/zhibeixiaoyu.git

# 推送代码
git branch -M main
git push -u origin main
```

> ⚠️ 如果提示登录，按窗口提示完成 GitHub 身份验证

---

## 第四步：在 Render 上部署

1. 打开 https://dashboard.render.com
2. 点击 **"New +"** → 选择 **"Web Service"**
3. 连接 GitHub：
   - 点击 "Connect account"，授权 Render 访问 GitHub
   - 选择 `zhibeixiaoyu` 仓库
4. 配置服务：

| 配置项 | 填写内容 |
|--------|----------|
| **Name** | `zhibeixiaoyu` |
| **Runtime** | `Python 3` |
| **Build Command** | `pip install -r zhibei_backend/requirements.txt` |
| **Start Command** | `cd zhibei_backend && python run_prod.py` |
| **Instance Type** | `Free` |

5. 添加环境变量（点击 "Advanced" → "Add Environment Variable"）：

| Key | Value |
|-----|-------|
| `DEEPSEEK_API_KEY` | 你的 DeepSeek API Key |
| `DEFAULT_AI_PROVIDER` | `deepseek` |
| `DEFAULT_AI_MODEL` | `deepseek-chat` |
| `RENDER` | `true` |

6. 点击 **"Create Web Service"**
7. 等待部署完成（首次约 3-5 分钟）

---

## 第五步：获取公共域名

## ⚠️ 重要注意事项

### 关于 Render 免费版的限制

| 限制 | 说明 |
|------|------|
| 冷启动 | 15分钟无访问后服务休眠，下次访问需等约30秒 |

所有用户共享你的 DeepSeek API Key，产生的 API 调用费用由你承担。
- DeepSeek Chat 模型价格约 ¥1/百万输入token，非常便宜
- 建议关注用量：https://platform.deepseek.com/usage

### 关于数据安全

- GitHub 仓库已设为 **Private**，API Key 不会泄露到代码中
- API Key 仅存储在 Render 的环境变量中，是加密的

## 🔧 常见问题

### Q: 部署失败，提示 "ModuleNotFoundError"
检查 `requirements.txt` 是否包含所有依赖，确认 Build Command 路径正确。

### Q: 网页打开但功能不正常
- 打开浏览器开发者工具（F12）→ Console 查看错误
- 确认环境变量 `DEEPSEEK_API_KEY` 已正确配置

### Q: 如何更新代码？
```bash
cd D:\dachuang\zhibeixiaoyu
git add -A
git commit -m "更新说明"
git push
```
Render 会自动检测到推送并重新部署。

### Q: 访问太慢怎么办？
考虑升级到 Render 付费版（$7/月），或使用 Railway（$5/月免费额度）。
